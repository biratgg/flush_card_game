## How to Run the Game

1. Clone the repository:

```
git clone https://github.com/biratgg/flush_card_game.git
```

2. Navigate to the project directory:

```
cd flush_card_game
```

3. Run the game:

```
python home.py
```

Make sure Python 3 is installed on your system.

---

## How the Game Works

* A deck of cards is shuffled
* Cards are dealt to the player
* Game logic evaluates the cards
* Results are displayed based on Flush card rules

*(Rules can be expanded or modified in `logic.py`)*
